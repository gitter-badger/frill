## Getting Started
Add [gulp](http://gulpjs.com/)
```bash
$ npm install -g gulp
```
then, clone or fork this repository:
```bash
$ git clone -b master --single-branch \
      https://github.com/nanopx/frill.git MyNewApp
$ cd MyNewApp && npm install
```
